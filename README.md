
# Did You Know

Een kleine Next.js-site die korte feiten toont — simpel, modern en makkelijk uitbreidbaar.

## Overzicht

Dit project is een hobby- of demonstratiewebsite gebouwd met Next.js en React. De focus ligt op een lichte, toegankelijke UI om korte 'did you know' feiten te tonen via herbruikbare componenten.

## Installatie & lokaal draaien

1. Repo klonen:

```
git clone <repository-url>
cd did-you-know-website
```

2. Dependencies installeren:

```
npm install
```

3. Development server starten:

```
npm run dev
```

De site is vervolgens beschikbaar op http://localhost:3000

## Scripts

- `npm run dev` — start development server
- `npm run build` — bouw voor productie
- `npm run start` — start de productie-server (na `build`)
- `npm run lint` — voer ESLint uit

Deze scripts corresponderen met de `package.json` in de projectroot.

## Mappenstructuur (belangrijkste bestanden)

- `app/` — Next.js app-map met pagina's en layout
- `components/FactWidget.tsx` — component voor feiten
- `public/` — statische assets
- `globals.css` — globale styles
- `next.config.ts`, `tsconfig.json`, `package.json` — projectconfiguratie
